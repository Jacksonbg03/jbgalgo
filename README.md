<h1 align="center">✨ JBGAlgo - Student Coding Platform ✨</h1>

An online judge and coding practice platform built for students to learn and improve their programming skills.

✨ Highlights:

- 🧑‍💻 VSCode-Powered Code Editor
- 🔐 Authentication via Clerk
- 🧭 Dashboard with Live Stats
- ⚙️ Secure Code Execution (JS, Python, Java)
- 🎯 Auto Verdict — Accepted / Wrong Answer based on test cases
- 🎉 Confetti on Success + Notifications on Fail
- 🧩 Practice Problems Page
- 🧠 Background Jobs with Inngest
- 🧰 REST API with Node.js & Express
- ⚡ Data Fetching & Caching via TanStack Query
- 🚀 Deployed on Render
